This plugin creates a widget that allows you to place your XBOX Live Gamertag on your sidebar.

== Description ==

I wish it were fancier, but it's a simple, stable plugin that adds a widget to Wordpress to display your XBOX 360 Gamertag on your sidebar.  

== Installation (Wordpress 2.5+) ==

1. Upload `xboxgamertag.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place your gamertag widget by clicking on 'Design' then 'Widgets'

== Installation (Wordpress 2.2 or 2.3) ==

1. Upload `xboxgamertag.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place your gamertag widget by clicking on 'Presentation' then 'Sidebar Widgets'

== Installation (Wordpress 2.0 or 2.1) ==

1. Upload `xboxgamertag.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Download <a href="http://automattic.com/code/widgets/">Automattic's Widgets plugin</a>, and place into your `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Upload `xboxgamertag.php` to the `/wp-content/plugins/widgets` directory
6. Activate the plugin through the 'Plugins' menu in WordPress
7. Place your gamertag widget by clicking on 'Presentation' then 'Sidebar Widgets'
